Beskrivning

Den här komponenten utgör ett api gentemot vgr:s Apelon-tjänst. Den avnänds för att ge användare av ifeed admin tillgång till listor av termer som används inom organisationen. Detta när man väljer sökvärden för vissa av filtren, exempelvis "Verksamhetskod enligt HSA", då dyker en rullgardinsmeny upp istället för ett vanligt inmatningsfält.

Koden i komponenten är mestadels genererad av en wsdl.
